# ✅ To-Do List App

Uma aplicação de gestão de tarefas com interface gráfica, desenvolvida em Python como projeto de portfólio.

![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python&logoColor=white)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-orange)
![License](https://img.shields.io/badge/License-MIT-green)

---

## 📸 Funcionalidades

- ✅ **Criar tarefas** com título e categoria
- 🏷️ **Categorias** — Pessoal, Trabalho, Estudo, Saúde, Outro
- ✔️ **Marcar como concluída** (ou desfazer)
- ✏️ **Editar** o título de qualquer tarefa
- 🗑️ **Apagar** com confirmação
- 🔍 **Pesquisa** em tempo real
- 🎛️ **Filtros** por categoria e estado
- 💾 **Persistência** em JSON (os dados guardam entre sessões)
- 📊 **Estatísticas** no topo (concluídas / pendentes / total)

---

## 🚀 Como executar

### Pré-requisitos
- Python 3.8 ou superior
- `tkinter` (já incluído no Python padrão)

### Passos

```bash
# 1. Clona o repositório
git clone https://github.com/SEU_USERNAME/todo-app.git
cd todo-app

# 2. (Opcional) Cria um ambiente virtual
python3 -m venv venv
source venv/bin/activate

# 3. Executa a app
python3 todo_app.py
```

> **Mac:** Se o tkinter não estiver disponível, instala com `brew install python-tk`
